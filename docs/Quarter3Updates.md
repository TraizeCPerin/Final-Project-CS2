Update:

This third quarter, we implemented the Maximum Height and Horizontal Range features of our Kinematics Calculator. The system reads the projectile’s initial velocity and launch angle from the kinematics JSON dataset and applies projectile motion formulas to compute the maximum height and total horizontal range.

The computed results are displayed clearly to the user based on the selected scenario ID. We also ensured that the program handles invalid inputs, such as incorrect scenario IDs or menu choices, by showing appropriate error messages. These features were tested using multiple scenarios to verify accuracy and functionality.
